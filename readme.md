# SignIn App

An Ionic Todo App in React

## Setup steps

1. Create `.env` file in root directory from `.env.sample`
2. Ask for values of `.env` from your team mate
3. Apply them by running `source .env`
4. Run `npm start` to start the app

## Features

- Login screen
- Registration
